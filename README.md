# Classic Todo list
A classic vanilla to do list with the classic add, delete, edit. 

Timer only in seconds purposefully because it's interesting to think about how far we have abstracted ourselves way from time that we don't notice how much of it we truly have to get the tasks we want to get done, done.

Run `git clone` to download this project to your local environment.
